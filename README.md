# SHAPE MYSTERY
## ゲームの内容
○△□の形が鍵・爆弾・ドアに対応しており、
正しい形を選んでゴールを目指すパズルゲーム。
一定時間ごとに形の対応がローテーションで切り替わるため、
常に頭の中で対応を追い続ける必要がある。
切り替わりのタイミングを予測して動いても、
ゴール直前に罠が待っているかもしれない。
全3ステージ、即死制。

## 操作説明
- 矢印キー、もしくはWASDキーでプレイヤーを移動する
- 鍵に対応する形に触れると鍵を入手できる
- 爆弾に対応する形に触れると即死してやり直しになる
- ドアに対応する形に触れると次のステージへ進む
